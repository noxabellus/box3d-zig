# box3d-zig

Minimal Zig wrapper for the glorious [box3d](https://box2d.org/documentation3d/index.html).

Builds from source, pinned by tag in [the zon file](./build.zig.zon).

This probably requires exactly [Zig version 0.16.0](https://ziglang.org/download/0.16.0/release-notes.html).

The bindings are still WIP as far as Zig-idiomaticity goes, but are fully usable.

I'll tag `v0.1.0` at some point in the near future (or whichever version upstream is on by then).
